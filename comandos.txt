1. Iniciar un repositorio en el proyecto desde VS Code

git init  # Inicializa un repositorio Git en la carpeta
git add .  # Agrega todos los archivos al repositorio
git commit -m "Primer commit"  # Guarda los cambios con un mensaje

git status # Para ver si hay cambios sin guardar en el repo local

git add .
git commit -m "Actualizacion de los ficheros de datos periodo 2000_2020"
git pull origin main --rebase # El git pull descarga los cambios de la rama main en el repositorio remoto y los aplica sobre tu rama local. Cuando otros han realizado cambios en el repositorio remoto, y deseas integrarlos con los tuyos antes de hacer un push



2. Vincular el repositorio local con GitHub

git remote add origin https://github.com/mgomezv26/App_Benthic_Organisms_Summary_Data.git
git branch -M main
git push -u origin main # git push, estÃ¡s subiendo los commits que has realizado en tu repositorio local a GitHub.
